# Contributing

Merci pour votre intérêt !

## Comment contribuer
- Forkez le dépôt et créez une branche dédiée.
- Faites des changements ciblés et testez-les si possible.
- Ouvrez une PR claire et concise.

## Style
- Restez cohérent avec le style existant du projet.
- Évitez les changements non nécessaires (formatage global, renommages, etc.).
